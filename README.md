# fx74-fx35
## English
Make Firefox 74 look like Firefox 3.5.

Codes from other sites are mentioned in the CSS code.

## 한국어
파이어폭스 74를 3.5처럼 보이게 만듭니다. 다만 도구 모음 색은 Windows 7/Vista 기준으로, 다른 윈도우 혹은 테마를 적용했거나 화면 배색을 변경해도 스타일이 반영되지 않습니다. 지못미 윈도우 XP.
